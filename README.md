# STL_Structures

In this project you will see a realization of different STL Structures such as:

- Binary Search Tree
- Dynamic Array
- Hash Table + String as hash
- Linked List
- Priority Queue
Also you can see an execution time for each structure expect LinkedList and DynamicArray
